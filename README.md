# Nested Grid & Flexbox - HW #4

The MMED-1056 HW#4 puts together four layout techniques for the hero section for the products hero section of the Orbitz website. The layout techniques include: `nested grid`, `flexbox`, `responsive images`, and the `order property`. These will help create a balanced, responsive and well-organized hero section for users to access across different devices.

## 💾 Installation
No installation required. Download the zip file and extract it.

## 🌐 Usage
Open the `index.html` file in the browser of your choice.
